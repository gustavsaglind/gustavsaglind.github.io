Animation & VFX - Building Skins


Tap the images below to view these in-game cosmetic skins that have been implemented in the top-grossing mobile game "Mobile Minigames". The 2D assets were created in Photoshop, then implemented inside Unity. Then the assets were animated using Unity's keyframe-based animation system and the VFX were created using Unity particle systems. Shaders were also used on some assets.
